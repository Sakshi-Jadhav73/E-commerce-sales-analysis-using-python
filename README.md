# 🛒 Sales and Profit Analysis Dashboard

## Project Overview
The Sales and Profit Analysis Dashboard analyzes a store’s sales and profit data using **Python**, **Pandas**, **NumPy**, and **Plotly** in a Jupyter Notebook.  
The dashboard provides **interactive visualizations** that help businesses understand:

- Monthly sales trends  
- Product category and sub-category performance  
- Customer segment contributions  
- Profitability and efficiency using sales-to-profit ratio

---

## Problem Statement
The main objectives are:

1. Calculate **monthly sales** and identify months with **highest and lowest sales**.  
2. Analyze sales by **product categories** and **sub-categories** to find top and bottom performers.  
3. Determine **monthly profit trends** and identify the month with **highest profit**.  
4. Analyze profit distribution across **categories** and **sub-categories**.  
5. Analyze sales and profit by **customer segments**.  
6. Calculate and analyze the **sales-to-profit ratio** for profitability insights.

---

## Tools and Technologies
- **Python** – Programming language for data analysis  
- **Pandas** – Data manipulation and aggregation  
- **NumPy** – Numerical operations  
- **Plotly** – Interactive visualizations  
- **Jupyter Notebook** – Interactive coding environment
